No bloco 3,eu aprendi...

  1. a inserir textos e imagens em uma página web;
  2. a inserir links externos e internos em uma página web;
  3. a alterar o fundo, a fonte e as cores de uma página web;
  4. a posicionar, redimensionar e formatar elementos de uma página web;
  5. a agrupar e condicionar elementos a obedecer determinadas funções em uma página web;
  6. a classificar e combinar elementos com mesmas propriedades em uma página web;
  7. a estilizar imagens e blocos de conteúdo com bordas e quebras de canto em uma página web;
  8. a separar elementos em "caixas" e fixá-las ou flutuá-las em uma página web;
  9. a hospedar códigos de HTML e CSS em uma página web através do GitHub Pages;
  10. a organizar meu código HTML de forma semântica e contextualizada.

Nos próximos blocos, eu espero aprender...

  • a criar uma plataforma de administração da página para leigos em programação;

  • a permitir que usuários possam personalizar elementos da minha página web;
  • a conectar bancos de dados síncronos com minha página web para gerir informações.

Meu aprendizado

No meu primeiro mês de estudos aquina Trybe, pude já aprender a lidar com o terminal e manipular meus diretórios e arquivos usando a linguagem do OS Unix, além de consumir conteúdos sobre HTML e CSS que me ensinaram a elaborar e estilizar uma página web de forma semântica para que seja acessível a qualquer tipo de usuário.

A metodologia Trybe

A metodologia da Trybe é a forma mais eficiente de se transmitir conhecimento que eu já vi. Nesse primeiro mês, além de aprender sobre o desenvolvimento web, interagi com pessoas para desenvolver minhas soft skills.
