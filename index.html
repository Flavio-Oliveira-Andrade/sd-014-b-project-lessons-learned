De recrutadora para recrutada

Os softwares de recrutamento não atendem aos PCDs.
Então decidi que vou criar meu próprio software de recrutamento que atenda a esses profissionais.

Ah, o Github...

Eu sou gateira raiz, a,o um gatinho! Mas o Git tá demais, é DR todo dia com os comandos.
